# Binary to Gray code converter
This project serves as an introduction to a new FPGA board i.e. EDGE ZYNQ SoC FPGA Development Board. A FSM that converts a binary input to its gray output has been designed using a State Machine. The State Machine was tested using Test bench before being deployed on the board. This project will later be used for ILA project in the future.
